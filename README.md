# Android Versions App

This Android Studio project is built in Java.

Features:
- RecyclerView list of Android versions.
- CardView item layout.
- ConstraintLayout inside each item.
- ImageView, code name, and version number for each row.
- Toast message when clicking any item.
- Alternate row colors for readability.
- Sort A-Z button.

How to run:
1. Open Android Studio.
2. Choose File > Open.
3. Select the AndroidVersions_Fixed folder.
4. Wait for Gradle Sync to finish.
5. Run the app on emulator or phone.
